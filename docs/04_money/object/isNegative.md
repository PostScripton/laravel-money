# `isNegative()`

checks whether the money's number is negative (less than zero).

## Methods

### `isNegative()`
**Returns**: `bool`

## Usage

```php
$m1 = money(-1000);
$m2 = money(1000);

$m1->isNegative(); // true
$m2->isNegative(); // false
```
